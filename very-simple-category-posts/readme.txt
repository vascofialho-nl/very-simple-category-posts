A lightweight plugin that displays all posts in the current category via shortcode, highlighting the current post. 

== Description ==

Use the shortcode [current_category_posts] inside any single post or sidebar text widget. It will display a list of posts from the same category and bold the current one.

- Automatically styled with inline CSS
- Great for internal navigation or reading suggestions
- No settings or configuration needed

== Installation ==

1. Upload the plugin files to the /wp-content/plugins/very-simple-category-posts/ directory, or install it via the WordPress plugin screen directly.
2. Activate the plugin through the 'Plugins' screen.
3. Use the shortcode [current_category_posts] inside any post or sidebar text widget.
